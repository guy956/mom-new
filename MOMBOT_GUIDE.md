## 🔧 מבנה טכני

### מחלקות עיקריות:

```dart
class AIChatScreen {
  // API Configuration
  static const String _geminiApiKey = 'AIzaSyDENOBbuoNc2XeyA7XRxVFQT51_TJrgD9s';
  static const String _geminiUrl = '...';
  
  // System Prompt
  static const String _systemPrompt = '''...''';
  
  // Conversation History
  final List<Map<String, dynamic>> _conversationHistory = [];
  
  // Methods
  Future<String> _callGeminiApi(String userMessage);
  void _sendMessage();
  void _scrollToBottom();
}

class _AIChatMessage {
  final String content;
  final bool isUser;
  final DateTime timestamp;
}
```

### תהליך עבודה:

1. **User sends message** → `_sendMessage()`
2. **Add to UI** → `_messages.add(_AIChatMessage(...))`
3. **Show typing indicator** → `_isTyping = true`
4. **Call Gemini API** → `_callGeminiApi(message)`
5. **Build context** → Add system prompt + conversation history
6. **Send to Gemini** → HTTP POST with JSON body
7. **Parse response** → Extract text from response
8. **Add to UI** → Show bot response
9. **Update history** → Save to `_conversationHistory`
10. **Scroll to bottom** → `_scrollToBottom()`

### API Request Structure:

```json
{
  "contents": [
    {
      "role": "user",
      "parts": [
        {"text": "System Prompt + User Question"}
      ]
    }
  ],
  "generationConfig": {
    "temperature": 0.8,
    "topP": 0.95,
    "topK": 40,
    "maxOutputTokens": 1024
  },
  "safetySettings": [
    {
      "category": "HARM_CATEGORY_DANGEROUS_CONTENT",
      "threshold": "BLOCK_MEDIUM_AND_ABOVE"
    }
  ]
}
```
