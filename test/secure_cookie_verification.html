ℹ️ About httpOnly and Client-Side Code

Important: True httpOnly cookies cannot be set from JavaScript. They must be set server-side in the HTTP response header:

Set-Cookie: session=abc123; HttpOnly; Secure; SameSite=Strict; Max-Age=86400

For Flutter web apps without a server, we use the __Host- prefix as defense-in-depth, which enforces:

  • Must have Secure attribute (HTTPS only)
  • Must have Path=/ (root path only)
  • Cannot have Domain attribute
