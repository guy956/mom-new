## 🎯 סיכום מהיר

**MOMIT** היא פלטפורמת רשת חברתית מקיפה לאמהות בישראל, עם דשבורד אדמין מתקדם המאפשר שליטה מלאה (100% no-code) בכל היבטי האפליקציה.

### ⚡ יכולות עיקריות:
- ✅ **25+ פיצ'רים** - מהפיד החברתי ועד מעקב תינוקות
- ✅ **AI MomBot** - עוזרת חכמה עם Google Gemini
- ✅ **דשבורד אדמין** - 17+ טאבים לניהול מלא
- ✅ **עריכה דינמית** - שינוי צבעים, תוכן, ניווט ללא קוד
- ✅ **Feature Flags** - הפעלת/כיבוי פיצ'רים בלחיצת כפתור
- ✅ **אבטחה ברמה גבוהה** - JWT, RBAC, Audit Log

## 👨‍💼 דשבורד אדמין מתקדם (17+ טאבים)

### 🎯 **יכולות עריכה (100% No-Code):**

#### 1. **Feature Control Center** ⭐
- הפעלה/כיבוי של **25+ פיצ'רים**
- שליטה מרכזית בכל האפליקציה
- סטטיסטיקות שימוש לכל פיצ'ר

#### 2. **Theme Studio** 🎨
- **כל הצבעים** ניתנים לשינוי
- בחירת גופן
- רדיוס פינות
- מצב כהה/בהיר
- תצוגה מקדימה חיה

#### 3. **Navigation Editor** 🧭
- סידור פריטי תפריט (drag & drop)
- הצגה/הסתרה של פריטים
- שינוי שמות ואייקונים
- הוספת פריטים מותאמים

#### 4. **Content Manager** 📝
- עריכת פוסטים, טיפים, הודעות
- עורך עשיר (Rich Text)
- העלאת תמונות
- אישור תוכן לפני פרסום

#### 5. **Content Scheduler** 📅
- תזמון פוסטים מראש
- תוכן חוזר (Recurring)
- לוח שנה ויזואלי

#### 6. **Analytics Dashboard** 📊
- משתמשים פעילים (יומי/שבועי/חודשי)
- שימוש בפיצ'רים
- ביצועי תוכן
- ייצוא דוחות PDF/CSV

#### 7. **User Management** 👥
- ניהול משתמשים
- חסימה/השעייה
- צפייה בפרופילים

#### 8. **Permissions Manager** 🔐
- 4 תפקידים: Super Admin, Admin, Moderator, Viewer
- 32 הרשאות נפרדות
- מטריצת הרשאות

#### 9. **Experts Management** 👩‍⚕️
- אישור מומחים
- ניהול תורים
- דירוגים וביקורות

#### 10. **Events Management** 📅
- יצירת אירועים
- ניהול רשימות משתתפים
- אישור אירועים

#### 11. **Marketplace Management** 🛒
- ניהול מודעות
- קטגוריות דינמיות
- דיווחים על מודעות

#### 12. **Reports & Moderation** 🚨
- טיפול בדיווחים
- מחיקת תוכן פוגעני
- צפייה בהיסטוריה

#### 13. **Dynamic Forms** 📋
- עריכת טופס הרשמה
- עריכת טופס SOS
- הוספת שדות דינמיים

#### 14. **Dynamic Sections** 🏗️
- ניהול אזורים במסך הבית
- סידור תוכן
- הוספת באנרים

#### 15. **Media Vault** 🗄️
- ניהול קבצים
- העלאה/מחיקה
- ארגון בתיקיות

#### 16. **Communication Center** 📢
- שליחת התראות push
- הודעות מערכת
- באנרים

#### 17. **Audit Log** 📋
- מעקב אחר כל פעולות האדמין
- מי עשה מה ומתי
- ייצוא לוגים

## 🔐 אבטחה

### מערכת הרשאות (RBAC):
| תפקיד | הרשאות |
|-------|--------|
| **Super Admin** | גישה מלאה לכל |
| **Admin** | ניהול משתמשים ותוכן |
| **Moderator** | מoderation ודיווחים |
| **Viewer** | צפייה בלבד |

### תכונות אבטחה:
- ✅ JWT Tokens עם Refresh Rotation
- ✅ Rate Limiting (הגבלת בקשות)
- ✅ Secure Cookies (httpOnly, secure, sameSite)
- ✅ CSRF Protection
- ✅ Input Validation
- ✅ Audit Log מלא
- ✅ Firestore Security Rules

## 🚀 התקנה והרצה

### דרישות:
```bash
Flutter 3.24+
Dart 3.0+
Firebase CLI
```

### התקנה:
```bash
# 1. Clone the repo
git clone https://github.com/guy956/mom.git
cd mom

# 2. Install dependencies
flutter pub get

# 3. Setup Firebase
firebase login
firebase init

# 4. Run locally
flutter run

# 5. Build for web
flutter build web --release --web-renderer html
```

---

## 📦 Deployment

### Cloudflare Pages:
1. בנה את הפרויקט: `flutter build web --release --web-renderer html`
2. העלה את `build/web` ל-Cloudflare Pages
3. הגדר דומיין: `momit.pages.dev`

### Firebase Hosting:
```bash
firebase deploy --only hosting
```
