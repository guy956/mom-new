# MOMIT - רשימת פיצ'רים מלאה לדשבורד אדמין

## 📱 פיצ'רי משתמש (25+)

### פיצ'רים מרכזיים:
1. **Feed** - פוסטים, לייקים, תגובות, שיתופים
2. **AI Chat (MomBot)** - צ'אטבוט עם Gemini
3. **SOS** - כפתור חירום לעזרה מיידית
4. **Tracking** - מעקב תינוקות (משקל, גובה, שינה, האכלה, אבני דרך)
5. **Events** - אירועים ומפגשים (סדנאות, וובינרים, ערבי נשים)
6. **Marketplace** - קנייה ומכירה
7. **Experts** - מומחים מאומתים
8. **Chat** - צ'אט בין משתמשות וקבוצות
9. **Photo Album** - אלבום תמונות
10. **WhatsApp** - אינטגרציה עם וואטסאפ
11. **Gamification** - מערכת נקודות ותגמולים
12. **Daily Tips** - טיפים יומיים
13. **Mood Tracker** - מעקב מצב רוח
14. **Notifications** - התראות push
